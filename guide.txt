4. Enable GitHub Pages for Web Display
This deploys the HTML reports (e.g., daily tech news summaries) to a live site like https://YOUR_USERNAME.github.io/TrendRadar.

In your forked repo: Go to Settings > Pages.
Set Source to "Deploy from a branch".
Branch: main | Folder: /(root).
Save. GitHub will build and host the site (takes 1-2 minutes).
Visit https://YOUR_USERNAME.github.io/TrendRadar to see the reports (updates after each workflow run).
The workflow auto-commits HTML files back to your repo, triggering Pages updates.

Rate limits: GitHub warns against <30-min runs; stick to hourly.
